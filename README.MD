# TS Config
Helper to quickly create `tsconfig.json` files

```sh
npx @mcha/cfiles
```

```sh
? Choose config file (Use arrow keys)
❯ .eslintrc.js
  tsconfig.json
  .prettierrc
  jest.config.js

# tsconfig.json has further options:
? Creating tsconfig.json for: (Use arrow keys)
❯ nextjs
  node
  react
  reactnative
```
